FRONTEND

En el proyecto de Java backend consiste en el desarrollo de una API Rest, la cual permite una interación con servicios web dentro de los temas que se pueden aprender están la utilización de una base de datos, utilizar Java Persistence API, a construir una API REST desde cero.

El Proyecto frontend es el complemento al proyecto anterior(backend) que consiste en darle una vista a la API a través de una aplicación web. Dentro de los temas a aprender son diseño web (html y css), Java Script,React.JS.
